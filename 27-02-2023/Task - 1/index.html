Selectors in CSS

CSS selectors define the pattern to select elements to which a set of CSS rules are then applied.

Universal Selector -

Selects all elements. Optionally, it may be restricted to a specific namespace or to all namespaces. For example - Text till now is styled using universal selector.

ID Selector

The CSS ID selector matches an element based on the value of the element's id attribute. In order for the element to be selected, its id attribute must match exactly the value given in the selector. For example - The current text is styled using id selector.

Element Selector

The CSS type selector matches elements by node name. In other words, it selects all elements of the given type within a document. For example - In current article tag we are using element selector as : artile { color : violet} to style this element.

Class Selector

The CSS class selector matches elements based on the contents of their class attribute. We add "." as a prefix before the class name to specify that it is a class selector. For example - The current paragraph element is styled using class selector.
