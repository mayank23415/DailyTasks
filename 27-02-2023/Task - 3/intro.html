Ways to insert CSS:

Inline CSS

We can apply CSS in a single element by inline CSS technique. The inline CSS is also a method to insert style sheets in HTML document. This method mitigates some advantages of style sheets so it is advised to use this method sparingly. If you want to use inline CSS, you should use the style attribute to the relevant tag. Click here to visit inline CSS implementation.

Internal CSS(Style tag)

In this we define the style tag inside the head tag. An internal style sheet may be used if one single HTML page has a unique style.Click here to visit site with internal css implementation.

External CSS

With an external style sheet, you can change the look of an entire website by changing just one file. We add this external css file by using either link tag where we give refrence of the css file we will be using or using import inside internal style tag.Click here to visit external css implementation.

Cascading Order

The order when more than one style is specified for an element in html document :

  1. Inline Style
  2. External and Internal Style
  3. Browser Default

Best practices

The External Style Sheet (using HTML link Tag) is the best method which is used to link the element. Maintaining and re-using the CSS file across different pages is easy and efficient. The link tag is placed in the HTML head element.
