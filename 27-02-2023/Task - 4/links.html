CSS Styling Links

Links can be styled with any CSS property (e.g. color, font-family, background, etc.).

With CSS, links can be styled in many different ways :

  • Unvisited Link : For unvisited link we use a:link. For eg - Click here to know about height and width property.
  • Visited Link : For visited links we use a:visited. For eg - Click here to know about height and width property.
  • Hover : Used for styling the link when it is hovered by mouse. For eg - Click here to know about height and width property.
  • Active Link : For active links we use a:active. For eg - Click here to know about height and width property.
