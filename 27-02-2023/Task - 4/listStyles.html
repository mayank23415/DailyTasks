CSS List Styles :

The CSS list properties allow you to:

  • Set different list item markers for ordered lists. For eg -
    • Coffee
    • Tea
    • Coca Cola
    • Coffee
    • Tea
    • Coca Cola
    1. Coffee
    2. Tea
    3. Coca Cola
    1. Coffee
    2. Tea
    3. Coca Cola
  • Add background colors to lists and list items
