preventDefault() and stopPropogation()

preventDefault() :

It is a method present in the event interface. This method prevents the browser from executing the default behavior of the selected element. This method can cancel the event only if the event is cancelable.

This event of clicking a link is prevented from executing by using preventDefault().

stopPropogation

This method is used to prevent the parent element from accessing the event. Basically, this method is used to prevent the propagation of the same event from being called. For eg, we have a button element inside a div tag and there is an onclick event on both of them, then whenever we try to activate the event attached to the button element, then the event attached to the div element also gets executed because div is the parent of the button element.

Here, after clicking on the button, both functions will be executed.

Here, after clicking on the button, only child function will be executed.
