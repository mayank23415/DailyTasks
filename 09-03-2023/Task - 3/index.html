ondblclick Event -

onmousedown & onmouseup Event -

The mouseDown() function sets the color of this text to red. The mouseUp() function sets the color of this text to blue.

onmouseenter & onmouseleave Event -

Color of text will be red when mouse enters the box and will be blue when mouse leaves the box.

onmouseover & onmouseout Event -

Color of text will be red when mouse is over the box and will be blue when mouse is out the box.
