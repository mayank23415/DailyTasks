In this html document we will demostrate the use of formatting elements

Bold formatting element

This element is used to make the text bold. This tag is used to make the text bold without extra importance.

Strong formatting element

This is used to define the text which have strong imortance. The content inside this tag is displayed in bold. It is usefull for screen readers.

Italic formatting element

This formatting element is used to make the text italic. This tag is used to indicate a technical term, a phrase from another language, a thought, a ship name ,etc.

Emphasized formatting element

This formatting element defines Emphasized text. The screen reader will pronounce the word with an emphasis, using verbal stress.

Small formatting element

This formatting element makes the text smaller.

Mark formatting element

The Mark element is used to highlight a particular section of the text.

Del formatting element

This element is used to define the text which has been deleted from a document.

Ins formatting element

This formatting element is used to define the text which has been inserted into a document.

Sub formatting element

This element stands for subscript and it is used to define subscript text. This type of text usually appears below the normal line, and is sometimes rendered in a smaller font. for example H2O

Sup formatting element

This element is used to define superscripted text. Superscripted text appear half a character above the normal line, and is sometimes rendered in a smaller font. For example WWW1
