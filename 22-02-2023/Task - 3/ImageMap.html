Image Mapping !

In image mapping an image is specified with certain set of coordinates inside the image which act as hyperlink areas to different destinations. It is different from an image link since in linking, an image can be used to serve a single link or destination whereas in a mapped image, different coordinates of the image can serve different links or destinations.

Elements required in Mapping an Image :

There are three basic html elements which are required for creating a mapped image.

  1. Map : It is used to create map of the image with clickable areas.
  2. Image : It is used for the image source on which mapping is done.
  3. Area : It is used within the map for defining clickable areas.
